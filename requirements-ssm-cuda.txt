# Optional CUDA/SSM dependency layer.
#
# Install requirements.txt first, then install this file on CUDA-devel systems
# with build isolation disabled so package setup can import the already
# installed torch CUDA build:
#
#   python -m pip install -r requirements.txt
#   python -m pip install --no-build-isolation -r requirements-ssm-cuda.txt
#
# This layer is expected to need a compatible NVIDIA driver, CUDA toolkit/nvcc,
# PyTorch CUDA build, and matching wheels or a working compiler toolchain. For
# source builds, set CUDA_HOME and TORCH_CUDA_ARCH_LIST for the target GPU. Set
# MAMBA_FORCE_BUILD=TRUE and CAUSAL_CONV1D_FORCE_BUILD=TRUE when a local build is
# required instead of an upstream wheel.

# pinned to the version used by the fresh-install and CUDA validation evidence.
# Mohawk's component import guard keeps the repository's top-level `utils`
# package ahead of third-party packages from this stack.
